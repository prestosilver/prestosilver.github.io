Hangover (gin2)

About

Hangover is a full rewrite of gin with the core goals:
  • Hangover will use opengl for graphics.
  • Hangover will have consistant naming.
  • Hangover will have a ui system.

Install

run: 

      $ nimble install http://github.com/bob16795/hangover
