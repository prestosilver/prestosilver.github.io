## About

Sandeee is a programming simulator with the following 3 goals.

- Nostalgic imagery
- Immense Immersion
- Different Simplicity

## Key Features

**Full Shell Scripting Language:**

Sandeee has a full shell with a custom scripting laguage, allowing you to automate simple tasks.

**Custom Executable Format:**

Sandeee allows you to compile custom executables which it runs in real time.

**3 Internal Programming Languages:**

Sandeee has 3 programming lanugages, allowing you to program however you want.
